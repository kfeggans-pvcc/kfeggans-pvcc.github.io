CHAPTER 6: Programing Languages

Machine Language

A computer programming language consisting of binary or hexadecimal instructions which a computer can respond to directly

Debugging

The process of identifying and removing error from computer hardware or software

Assembler

A program for converting instructions written in low-level symbolic code into machine code

Assembly Language

A low-level symbolic code converted by an assembler

Translator

A translator or programming language processor is a generic term that can refer to a compiler, assembler, or interpreter. Anything that converts code from one language to another

Compiler

A special program that translates a programming langauage's source code into a maching code, bytecode or another programming language

Interpreter

Transforms or interprets a high-level programming code into code that can be understood by the maching (machine code) or into an intermediate language that can be easily executed as well

Cross-platform software

A type of application/program/software/ that works on various operating systems or devices, which are often called platforms

Natural Language

Refers to the branch of computer schience-and more specifically, the branch of artificial or AI-concerned with giving computers the ability to understand text and spoken words in much the same way human beings can

Formal Language

A set of strings of symbols drawn from a finite alaphabet

Machine independence

A term applies to software that is not dependent on the properties of a particular machine, and can therefore be used on any machine

High-level programming language

Has a signigicant abstraction from the details of computer operations. It is designes to be easily understood by humans and for this reason they must be translated by another software

Object-oriented programming (OOP)

A computer programming model that organizes software design around data, or objects, rather than functions and logic

Scripting Language

A programming language that employs a high-level contruct to unterpret and execute one command at a time

Data Types

Integer Whole numbers represented as binary values. Most programming languages provide a data type called "integer", often called "int" for short Float A data type composed of a number that is not an integer String Used for data values that are made up of ordered sequences of characters

Primitive data type

A set of basic data types from which all other data types are constructed

Data structures

A specialized format for organizing, processing, retrieving and storing data Pything Collection data structures List An abstract data type that represents a finite number of ordered calues, where the same value may occur more than once Tuple An ordered sequence of elements of different data types, such as integer, float, string, list or even a tuple Set An abstract data type that will store an enordered colllection of unique values Dictionary An abstract type that defines an unordered collection of data as a set of key-value pairs

Function

A "chunk" of code that you can use over and over again, rather than writing it out multiple times

Local variable

A variable that is only accessible within a specific part of a program

Global Variable

Variable that are accessible globally, or everywhere throught the program

Source Program

The list of human-readable instructions that a programmer writes-often in a word processign program-when he is developing a program

Object program

A programming paradigm based on the concept of "objects", which may contain data, in the form of fields, often known as attributes; and code, in the form of proceduresm often known as methods
