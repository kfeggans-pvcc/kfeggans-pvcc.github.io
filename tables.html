TERMS DEFFINITIONS
operating system he software that supports a computer's basic functions
Job queue a data structure maintained by job scheduler software containing jobs to run
FIFO a method of queuing or memory management
Interactive Processing he process of interacting directly with a program and/or data for immediately modifying, retrieving and/or displaying information
Time-sharing a type of operating system in which the user can perform more than one task and each task gets the same amount of time to execute
Multiprogramming the technique of utilizing several programs concurrently in a single computer system via multiprocessing
Application Software An application program (software application, or application, or app for short) is a computer program designed to carry out a specific task other than one
System Software a type of computer program that is designed to run a computer's hardware and application programs
Utility Software helps users configure, analyse, optimise and maintain a computer
Graphical User interface (GUI) a visual way of interacting with a computer using items such as windows, icons, and menus, used by most modern operating systems
Shell a UNIX term for the interactive user interface with an operating system
Kernel he core that provides basic services for all other parts of the OS
Device manager a Control Panel applet within Windows operating systems
File Manager A file manager or file browser is a computer program that provides a user interface to manage files and folder
Memory Manager a method in the operating system to manage operations between main memory and disk during process execution.
Virtual Memory a memory management technique where secondary memory can be used as if it were a part of the main memory
Boot strapping get (oneself or something) into or out of a situation using existing resources.
ROM Read-only memory, or ROM, is a type of computer storage containing non-volatile, permanent data that, normally, can only be read, not written to
Deadlock a state of inaction or neutralization resulting from the opposition of equally powerful uncompromising persons or factions
Spooling send (data that is intended for printing or processing on a peripheral device) to an intermediate store.
Multi-Factor Authentication is an authentication method that requires the user to provide two or more verification factors to gain access to a resource such as an application, online account, or a VPN
Super User(or ADMIN) superuser is a special user account used for system administration
